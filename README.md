## 🧑‍💻 소개

### **사용자 경험을 최우선으로 하는 프론트엔드 개발자입니다.**

> 직관적인 UI와 최적화된 UX 구현에 집중하며, 사용자 니즈를 반영한 더 나은 경험을 추구합니다.
> 단순히 기능을 만드는 것이 아니라, **사용자가 만족하는 서비스를 개발하는 것**이 저의 가장 큰 동기부여 요소입니다.
